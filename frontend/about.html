About

what_it_isAn SSH wargame. You connect to a real Linux machine running in an isolated container, find a hidden password, and use it to log in as the next user. Repeat until done.
how_it_worksClick "Start session" on the home page. A container spins up and you get an SSH command. Each level user has a clue or file that leads to the next level's password. Sessions expire after 2 hours.
who_its_forPeople learning Linux, command-line navigation, and basic security concepts. No prior hacking experience required — just curiosity and a terminal.
isolationEach session runs in its own container with strict resource limits. No egress network access. You can't reach other users' machines or the internet from inside.
no_accountSessions are anonymous and stateless. Nothing is saved between sessions. Your progress resets when the container dies.

How to connect

You need an SSH client. On Linux and macOS it's built in. On Windows, use PowerShell, Windows Terminal, or PuTTY.

The starting password is always level0. Everything after that you find yourself.
