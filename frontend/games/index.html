Games

Each game is a self-contained series of levels running on a shared SSH server. No account required — just connect and start playing.

Available

Thief 9 levels · Linux file navigation · find hidden passwords across the filesystem

How it works

connectSSH in as the first user of a game — credentials are on each game's page
findLocate the password hidden somewhere on the machine
advanceSSH in as the next user using the password you found
repeatKeep going until you reach the final level
