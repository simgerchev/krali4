v1.0.1 2026-04-16 [latest]

  • stack page — restructured into Backend / Challenge containers / Infrastructure / Security / Frontend sections

v1.0.0 2026-04-16 [minor]
  • platform rebuilt around real SSH — browser terminal removed
  • on-demand containers per session — one isolated machine per player
  • 5 Bandit-style levels: file reading, tricky filenames, hidden files, binary vs text
  • passwords generated randomly at container start — no shared answers
  • session launcher on home page — SSH command shown after "Start"
  • 2-hour TTL with countdown timer and manual stop
  • FastAPI backend with Docker SDK for container lifecycle management
  • orphan cleanup on backend restart
  • hardened containers: cap_drop ALL, no-new-privileges, resource limits, no egress
  • Kubernetes manifests: RBAC, NetworkPolicy, Pod/NodePort template
  • nginx proxy serves frontend and routes /api/* to backend
  • rate limiting: 3 sessions/hour per IP in production
